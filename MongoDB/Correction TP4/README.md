# TP4 - FastAPI + MongoDB (sample_mflix)

## Installation

```bash
pip install -r requirements.txt
```

## Lancer l'API

```bash
uvicorn main:app --reload --port 4000
```

## Lancer le chatbot

```bash
python chatbot.py
```

## Tester dans le navigateur

- http://localhost:4000
- http://localhost:4000/docs
